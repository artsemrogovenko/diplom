Параметры запуска для среды разработки IntelliJ idea всех приложений в папке .run

localhost:8084 - сервис для выбора задач на исполнение
localhost:8083 - загрузка файлов схем
localhost:8082 - составление спецификации
localhost:8081 - сайт склада
localhost:8085 - сайт для заполнения формы выпускаемой продукции

localhost:8084 - сервис для выбора задач на исполнение требует авторизации для принятия задачи.
Попасть на форму авторизации нужно нажать на кнопку Login Page . ввести admin admin.
далее чтобы создать исполнителя нужно зайти в Admin panel. Заполнить поля и нажать зарегистрировать.
Далее администратор может выйти. Зарегистрированный пользователь пишет свое имя и фамилию через пробел в одной строке.
